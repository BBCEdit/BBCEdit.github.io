0.34.0

New Features

  • Added option to purge broken links from the recent files list.
  • Added option to check for possible UTF-8 encoding and corrupted strings/comments when loading a program.
  • Added option to disable syntax colouring (might give a slight performance boost on slow systems).

Cosmetic

  • Changed .BBC file icon to something more recognisable using the icon supplied with BBCSDL.

Bug Fixes

  • IMPORTANT! Fixed tokenise routine so that it does not corrupt line continuation comments that contain a period.
  • IMPORTANT! Fixed comments not being de-tokenised correctly, resulting in data corruption when comments contained ANSI/UTF-8 characters and one or more of the bytes in the character encoding matched a keyword byte token.
  • Fixed edit line changes being forgotten when opening the function list or making a 'Go To...' jump.
  • Fixed 'Go To...' dialogue not opening after a line selection.
  • Fixed variable names beginning with 'DEF' ending up in the function list.
  • Fixed line indentation not responding when certain changes were made, followed by cursor key up or down.
  • Fixed bug that prevented data recovery from working properly on next launch following a fatal error.

0.33.0

New Features

  • Added option to clear recent files list.
  • Added option to show full path in recent files list.
  • Added links to useful resources in help menu.

Structural

  • Recent files list now stored in _options.ini

0.32.4

Bug Fixes

  • IMPORTANT! Fixed save options when adjusting menu settings in file dialogue.
  • Edit line highlight fix when using Ctrl+Left-Arrow at carrot index zero.
  • Go To option edit box value auto selected correctly when dialogue is opened.
  • Status Bar view option stored correctly in _options.ini file.

Structural

  • Minor streamlining of code, including the removal of the file load progress dialogue.

0.32.3

New Features

  • Updated function list: enhanced menu navigation, implemented alphabetical sorting, and option to strip keywords (FN / PROC) from names.
  • Implemented 'Go To' option.
  • Implemented 'Go Back' option, available after changing the edit line using the function list or 'Go To' options.

Structural

  • BBCEdit program files moved into 'BBCEdit' folder to minimise footprint in the BBC BASIC for SDL tools/ folder.
  • Changed from .dat to .ini to store program configuration.
